# Memory System Skill

Use the engineering memory before modifying code.

## Workflow

1. Read the global brain indexes.
2. Identify the active project.
3. Load project memory.
4. Start with distilled context.
5. Load supporting notes only when necessary.
6. Inspect real code.
7. Implement changes.
8. Document project-specific learnings.
9. Promote reusable learnings into cross-project memory when appropriate.
