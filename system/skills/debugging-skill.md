# Debugging Skill

## Purpose

Solve bugs using structured context, root cause analysis, and durable documentation.

## Workflow

1. Load relevant rules and debugging patterns.
2. Load project-specific distilled context.
3. Compare expected behavior with actual behavior.
4. Compare working and broken flows.
5. Identify root cause.
6. Fix the right layer.
7. Validate.
8. Document the fix.
9. If the lesson is reusable, update cross-project memory.
