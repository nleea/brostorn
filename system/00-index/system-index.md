# System Index

This index describes the structure and purpose of the global engineering memory.

## Sections

### 01-global-patterns
Reusable technical patterns shared across projects.

### 02-global-skills
Reusable agent skills and ways of working.

### 03-global-commands
Standard workflows for common tasks such as debugging and refactoring.

### 04-global-rules
Guardrails for how agents should work.

### 05-cross-project-memory
Lessons, recurring bugs, reusable fixes, and anti-patterns discovered across projects.

## Usage

Use this system before entering project-specific documentation.

This system provides:
- reusable engineering knowledge
- operational workflows
- cross-project lessons
- active-project guidance that complements `projects/<project>/project.config.json`
