# Cloudflare

## Uso actual
- Cloudflare R2 para archivos, fotos, evidencias y PDFs.

## Variables relevantes
- `R2_ACCOUNT_ID`
- `R2_ACCESS_KEY_ID`
- `R2_SECRET_ACCESS_KEY`
- `R2_BUCKET_NAME`

## Problemas comunes
- Credenciales inválidas.
- Bucket incorrecto.
- Permisos S3 incompatibles.
