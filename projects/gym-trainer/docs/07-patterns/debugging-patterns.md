# Debugging Patterns

- Validar primero estado de sesión cuando hay `401`.
- Revisar claves lógicas de upsert cuando aparecen duplicados.
- Verificar locks y sincronización multi-tab en problemas de auth.
- Confirmar credenciales R2 y bucket cuando fallan uploads.
- Revisar migraciones actuales antes de asumir problemas de schema.
