# ADR-003 MCP Server

## Status
Unverified in project runtime docs

## Decision
Los `.md` raíz analizados no documentan un MCP server como parte del producto.

## Consequence
No se incorpora a la arquitectura funcional del sistema hasta tener documentación o código fuente asociado.
